API Films

Documentation

Cette API permet d'afficher, d'ajouter, de modifier et de supprimer des films de la base de données. Les routes sont accessibles à partir de l'url suivante : https://cadriel-front.onrender.com/films

Routes publiques

1. Récupérer la liste de tous les films

GET /api/films/ : https://cadriel-front.onrender.com/films

Retourne la liste de tous les films. Vous pouvez trier les films par année, par titre et par réalisateur de façon ascendante ou descendante à l'aide de paramèetres dans l'URL. Les paramètres possibles sont les suivants :

    tri
  • annee (par défault)
  • titre
  • realisation
    ordre
  • asc (par défault)
  • desc
    limite
  • un nombre en entier

Exemple : https://cadriel-front.onrender.com/films?trierPar=titre&ordre=desc&limite=5

2. Récupérer un film spécifique

GET /api/films/:id : https://cadriel-front.onrender.com/films/{id}

Le paramètre requis : id (l'identifiant du film à récupérer).

Exemple : https://cadriel-front.onrender.com/films/5105ghJSiQgsN8SsUaCv

3. Ajouter un nouveau film

POST /api/films/ :https://cadriel-front.onrender.com/films

Ajoute un nouveau film à la liste de films. Paramètres requis : un objet film dans le corps de la requête. La réussite de la requête retourne l'id du film ajouté.

Exemple : const film = { titre: "Titre du film", description: "Description du film", annee: 2024, realisation: "Nom du réalisateur", genres: ["Genre 1", "Genre 2"], titreVignette: "Titre de la vignette.jpg" }

4. Modifier un film

PUT /api/films/:id : https://cadriel-front.onrender.com/films/{id}

Modifier le film ayant l'identifiant :id. Les paramètres requis : l'id du film à modifier, un objet film dans le corpos de la requête.

Exemple : https://cadriel-front.onrender.com/films/gN0yc4PyLOsj8GA09tVr

const film = { titre: "Titre du film", description: "Description du film", annee: 2024, realisation: "Nom du réalisateur", genres: ["Genre 1", "Genre 2"], titreVignette: "Titre de la vignette.jpg" }

5. Supprimer un film

DELETE /api/films/:id : https://cadriel-front.onrender.com/films/{id}

Le paramètre requis : id (l'identifiant du film à récupérer).

6. S'inscrire au site

POST /api/utilisateurs/inscription : https://cadriel-front.onrender.com/utilisateurs/inscription

Ajoute un nouvel utilisateur à la liste des utilisateurs. Paramètres requis : un objet utilisateur dans le corps de la requête. La réussite de la requête retourne l'id de l'utilisateur ajouté.

Exemple : const utilisateur = { courriel: "utilisateur@email.com", mdp: "motDePasse" }

7. Se connecter à son compte

POST /api/utilisateurs/connexion : https://cadriel-front.onrender.com/utilisateurs/connexion

Connecte un utilisateur. Paramètres requis :un objet utilisateur dans le corps de la requête.

Exemple : const utilisateur = { courriel: "utilisateur@email.com", mdp: "motDePasse" }
